How to use git

Thu 07 October 2021
By Lucas Zeng
Category: Blog, Git

git workflow in a nutshell:

  • git checkout -b <feature-branch>

  • work on it

  • git add ... && git commit -m "..."

  • git push -u origin <feature-branch>

  • chat with colleagues and let them merge it for you
